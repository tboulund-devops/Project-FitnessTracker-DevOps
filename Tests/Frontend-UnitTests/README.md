# Frontend Unit Tests

This test package runs unit tests for the React + Vite frontend from `Frontend/src`.

## Run

```powershell
cd C:\Users\mbert\RiderProjects\Project-FitnessTracker-DevOps\Tests\Frontend-UnitTests
npm install
npm run coverage
```

Coverage thresholds are configured to require 100% for lines, branches, functions, and statements on the explicitly listed frontend files in `vitest.config.ts`.
